# M5PaperS3 SmartCard

Просмотрщик PNG-изображений для **M5Stack PaperS3** (E-Ink 4.7", ESP32-S3) с SD-карты.
Удобен для использования как «карточки» — шпаргалки, QR-коды, схемы, заметки, иконки.

![platform](https://img.shields.io/badge/platform-M5PaperS3-blue)
![framework](https://img.shields.io/badge/framework-Arduino-teal)

---

## ✨ Возможности

- 📂 Автоматическое сканирование SD-карты на PNG-файлы (с подпапками, до 5 уровней).
- 📃 Постраничное меню с фиксированной высотой кнопок и центрированием.
- 🖼 Полноэкранный показ PNG c автомасштабированием по ширине/высоте экрана.
- 🔄 Независимый поворот **меню** и **изображений** (0° / 90° / 180° / 270°).
- 🔋 Индикатор заряда, напряжения и состояния зарядки в верхнем баре.
- 💾 Сохранение настроек в NVS (Preferences).
- 🔉 Тактильный звуковой отклик при нажатии (короткий «бип»).
- ⚡ Адаптивный режим E-Ink: быстрый (`epd_fast`) для меню, качественный (`epd_quality`) для картинок.

## 📦 Зависимости

Установить через **Arduino IDE → Library Manager**:

- **M5Unified** (актуальная)
- **M5GFX** (актуальная)
- **SD** (входит в ESP32 core)
- **Preferences** (входит в ESP32 core)
- **SPI** (входит в ESP32 core)

Плата: **M5Stack-PaperS3** в Boards Manager (M5Stack ESP32).

---

## 🚀 Запуск

1. Установи Arduino IDE 2.x и ESP32-плату от M5Stack.
2. Установи библиотеки `M5Unified` и `M5GFX`.
3. Клонируй / скачай этот репозиторий и открой `M5PaperS3_SmartCard.ino`.
4. Выбери плату **M5PaperS3** и порт.
5. Положи на microSD любые `.png` (можно по папкам).
6. Скомпилируй и залей.

## 🎮 Управление

### Главный экран (список файлов)

- **Тап по файлу** — открыть.
- **Свайп влево** — следующая страница.
- **Свайп вправо** — предыдущая страница.
- Кнопка **Settings** (на 1-й странице) или **< Prev** (на остальных).
- Кнопка **Next >** или **Rescan** в конце списка.

### Просмотр изображения

- **Тап в любом месте** — вернуться в меню.

### Настройки

- **Menu rotation** — поворот меню (0/90/180/270°).
- **Image rotation** — поворот изображений независимо от меню.
- **Debug info** — показывать имя файла поверх картинки.
- **Rescan SD** — перечитать карту.
- **< Back to menu** — назад.

## ⚙️ Конфигурация

Все ключевые параметры — в `Common.h`:

```cpp
#define MAX_PNG_FILES     100   // лимит файлов в индексе
#define FILES_PER_PAGE    8     // файлов на страницу
#define BTN_HEIGHT        80    // высота кнопки, px
#define BTN_GAP           8     // зазор между кнопками
#define BATTERY_UPDATE_MS 60000 // период обновления батареи
#define FLICK_THRESHOLD   50    // мин. длина свайпа, px
#define SPEAKER_VOLUME    180   // громкость "бипа" 0..255
```
